Experience Designer workspaces
(IBM)

I designed the user interface of a workspace card system to allow marketing teams to track their work and approvals. Considerations such as filtering, statuses, and access to most important tasks were taken in this design.

MyRecipes mobile application
(academic project)

I was given the opportunity for a final project during my Mobile Application Development Certificate to create an original app. I decided to create an app that allows a user to save recipes on their device and access them at a later time. I designed this mockup flow as a blueprint for the app. My goal was to create a straightforward and intuitive flow that gave the user comprehensive options for things such as adding ingredients to a recipe.

Experience Designer storyboards
(IBM)

Experience Designer storyboards are designed to allow marketers to easily and visually put together marketing campaigns. The trick of this design was to combine a myriad of options such as channels, campaign stages, budgeting, marketing goals, and segments into a useable and powerful interface. This application also allowed for simultaneous editing and included collaborative tools such as commenting capabilities.

Sterling Store order picker
(IBM)

I helped design the UI and identify usability issues for a suite of order fulfilment tools by IBM Sterling Store. One of the interesting challenges of this design was its need to be fully responsive and support a variety of different devices. It also needed to receive input from various sources such as scanners, user taps, and computer mice. Workflows that could be picked up and completed by multiple store associates needed to be carefully considered.
