TechBE company website
(academic project)

I was given the opportunity for a final project during my Mobile Application Development Certificate to create an original responsive, accessible website. My Recipes mobile app
(academic project)

I was given the opportunity for a final project during my Mobile Application Development Certificate to create an original mobile app. I decided to create an app that allows a user to save recipes on their device and access them at a later time.

The app includes a SQLite database that I designed, a streamlined visual aesthetic, and a clear user workflow. The version of the app that you can try out by clicking the button below was coded for Android.

Emma Maxim ePortfolio
(personal project)

I designed this portfolio website (that you are currently visiting!) with specific requirements in mind. I wanted to challenge myself to build something visually appealing with minimal reliance on images and logos. I used colour, fonts, space, animation, and geometry to achieve my vision.

This website has been coded using HTML, CSS, and vanilla JavaScript, plus a touch of jQuery and PHP. I used Bootstrap for certain responsive elements, most notably the navigation bar.
